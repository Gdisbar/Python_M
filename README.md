
## Summary Comparison
========================================================
| Model | Implementation | Task Switching | Best For |
|---|---|---|---|
| asyncio | Single Thread | Cooperative (await) | High-concurrency I/O |
| Multithreading | Multiple Threads | Preemptive (OS-driven) | I/O with shared state |
| Multiprocessing | Multiple Processes | Parallel (on CPU cores) | CPU-intensive math |
